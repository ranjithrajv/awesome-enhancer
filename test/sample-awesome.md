# Awesome JavaScript Libraries

A curated list of awesome JavaScript libraries and resources.

## HTTP Clients

- [axios](https://github.com/axios/axios) - Promise based HTTP client
- [node-fetch](https://github.com/node-fetch/node-fetch) - A light-weight module that brings window.fetch to Node.js

## CLI Tools

- [commander](https://github.com/tj/commander.js) - node.js command-line interfaces made easy
- [inquirer](https://github.com/SBoudrias/Inquirer.js) - A collection of common interactive command line user interfaces

## Web Scraping

- [cheerio](https://github.com/cheeriojs/cheerio) - Fast, flexible & lean implementation of core jQuery
- [puppeteer](https://github.com/puppeteer/puppeteer) - Headless Chrome Node.js API

## Markdown

- [remark](https://github.com/remarkjs/remark) - Markdown processor powered by plugins
- [marked](https://github.com/markedjs/marked) - A markdown parser and compiler

## Testing

- [jest](https://github.com/facebook/jest) - Delightful JavaScript Testing
- [mocha](https://github.com/mochajs/mocha) - Simple, flexible, fun javascript test framework

## Utilities

- [lodash](https://github.com/lodash/lodash) - A modern JavaScript utility library
- [date-fns](https://github.com/date-fns/date-fns) - Modern JavaScript date utility library
